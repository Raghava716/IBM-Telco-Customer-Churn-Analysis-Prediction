# Customer Churn Analysis & Prediction

An end-to-end data analytics and machine learning project focused on identifying customers at risk of churn and providing actionable insights to improve customer retention.

## 🚀 Project Overview

Customer churn is one of the biggest challenges faced by subscription-based businesses. This project analyzes customer behavior, identifies churn patterns, and builds a predictive model to help businesses proactively retain customers.

## 📊 Key Features

* Cleaned and preprocessed 7,000+ customer records using **Pandas**.
* Performed **Exploratory Data Analysis (EDA)** to identify churn patterns and customer behavior trends.
* Wrote optimized **SQL queries** using joins, subqueries, and window functions for customer segmentation and business insights.
* Built an **XGBoost Classification Model** achieving **85%+ accuracy** in predicting customer churn.
* Developed an interactive **Power BI Dashboard** to track churn metrics and revenue impact.

## 🔍 Key Insights

* Customers with month-to-month contracts are more likely to churn.
* Customers with shorter tenure have a higher probability of leaving.
* Electronic check users show higher churn rates compared to other payment methods.
* Identifying high-risk customers enables proactive retention strategies.
